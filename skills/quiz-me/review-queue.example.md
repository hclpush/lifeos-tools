# Learning Review Queue

Spaced-repetition ledger for the Daily Learning Loop. Written by `quiz-me`. Copy this to `<VAULT>/000_Agent/learning/review-queue.md`.

**Cadence:** a concept scored **< 7/10** enters the queue and resurfaces at **+1, +3, +7 days**. Three clean passes (≥ 7) → `retired` (move to the bottom log). A concept scored ≥ 7 on first test never enters the queue.

**Columns**
- `next review` — date this concept is due (`YYYY-MM-DD`)
- `streak` — consecutive passes (≥7); resets to 0 on a miss
- `stage` — `1d` → `3d` → `7d` → `retired`
